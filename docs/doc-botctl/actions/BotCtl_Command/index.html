BotController - Command

Die BotController Command Funktion, erlaubt es dir Ingame Befehle an den Server zu senden.

info

Du kannst damit keine Befehle an andere Installierte Addons oder Mods senden.

Permissions​

Der User benötigt die Permission main.botctl.command um diesen Befehl ausführen zu können. Weitere Infos zu Permissions findest du hier.

Funktion​

Mit einem Klick auf dem Button Command öffnet sich ein Formular, gebe hier den Befehl ein.

Wichtig: ohne / am Anfang des Befehls.

Befehls Blackliste​

Wenn du Befehle blockieren willst, die man über den Command Button nicht ausführen darf, kannst du die Regex-Blackliste in der dynamicconfig.jsonnutzen. Die Blackliste prüft ob der gesendete Befehl einem der RegEx in dem JsonArray ommand_regex_filter_array entspricht und blockiert diesen Befehl.

Beispiel:​

In diesem Beispiel werden die Befehle pay, trust und add blockiert.

"command_regex_filter_array": [
  "\\bpay\\b",
  "\\btrust\\b",
  "\\badd\\b"
]

Regex kannst du einfach mit ChatGPT erstellen, weitere Infos findest du hier
