2. Discord-Bot erstellen & einrichten

Discord-Bot erstellen​

  1. Gehe auf die Discord Developer Seite und logge dich ein.

  2. Klicke auf New Application (rechts oben) und gib deinem Bot einen Namen.

  3. Klicke auf Create.

  4. Klicke im Linken Menü auf Installation.

  5. Wähle bei Install Link den Wert none aus.

  6. Klicke auf Save Changes.

  7. Klicke im Linken Menü auf Bot. Scrolle runter und setze ``Public Botauf aus und alle 3Privileged Gateway Intents` auf an. (Siehe Bild)

  8. Klicke auf Save Changes.

  9. Klicke im Linken Menü auf OAuth2. Scrolle runter und setze OAuth2 URL Generator auf bot und scrolle nach unten zu Bot Permissions. Wähle die Permissions aus, die dein Bot benötigt. (Siehe Bild)

Du kannst dem Bot Admin geben oder individuelle Permissions. Der Bot benötigt mindestens die Permissions Read Messages, Send Messages, Embed Links, Attach Files, Read Message History, Use External Emojis, Add Reactions.

  1. Scrolle ganz nach unten und kopiere die generated URL und füge sie in deinen Browser (neuer Tab) ein. Wähle den Server aus, auf dem der Bot hinzugefügt werden soll und klicke auf Authorize.

  2. Klicke nun im Links im Menü auf Bot und drücke den Blauen Button bei Reset Token. Kopiere den Token!
